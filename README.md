### 介绍
欢迎来到qt_tutorial,这里是Qt Framework相关教程代码仓库，致力于带你从入门到"放弃"(doge)。 教程按照**模块**、**继承结构**等思路进行深入。
> 视频链接: https://www.bilibili.com/video/BV1824y1d7kM

> PPT/开发环境/软件网盘链接: https://pan.baidu.com/s/1eD2FhFdCkXYc5hoVn6QHRw?pwd=jb7m

> Github仓库: https://github.com/zzzcb/qt_tutorial

> Gitee仓库: https://gitee.com/changbozhang/qt_tutorial

### 工程代码说明
- 构建工具采用CMake(>=3.16)
- Qt版本采用Qt6.7.2
- VS版本采用VS2022
